Description

Attributes

The parent index section must have a class of "index".

Additional Constraints

Indexterm inner html may be popluated with a number of different things. You may choose to auto-generate the text via the CSS, using either the page number where the term occurs (appropriate for fixed-page presentations, e.g. print and PDF), or establishing a counter in the CSS and numbering each occurrance of a term sequentially, e.g.: An Index Term, 1, 2. See default.css for examples of using the target page number or a new counter. You may also populate the inner html with arbitrary text, e.g., the title of the section in which the term occurs.
